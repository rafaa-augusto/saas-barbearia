💈 SAAS Barbearia

Sistema/landing page desenvolvido para uma barbearia, com foco em apresentar serviços, cortes e facilitar o contato com clientes.

O projeto simula a interface de um SaaS para barbearias, podendo ser usado como base para sistemas de agendamento ou páginas institucionais.

🚀 Tecnologias utilizadas

HTML

CSS

Sass

📂 Estrutura do projeto
```
SAAS-BARBEARIA
│
├── assets
│   ├── bg.svg
│   ├── corte1.png
│   ├── corte2.png
│   ├── corte3.png
│   ├── images.svg
│   ├── logo.svg
│   ├── logo-black.svg
│   └── whatsapp.svg
│
├── partials
│   ├── colors.scss
│   ├── header.scss
│   └── layout.scss
│
├── index.html
├── style.scss
└── style.css
```


📌 Organização

assets/ → imagens e ícones utilizados no site

partials/ → arquivos SCSS separados por responsabilidade (cores, layout, header)

index.html → página principal

style.scss → arquivo principal de estilos

🎯 Funcionalidades

Layout moderno para barbearia

Apresentação de cortes

Design responsivo

Integração visual com contato via WhatsApp

Estrutura escalável com SCSS
